# Cognizant Java Full Stack Engineer (FSE) – Deep Skilling Program

## 📌 About

This repository contains my weekly hands-on exercises, coding assignments, and mini-projects completed as part of the **8-Week Java Full Stack Engineer (FSE) Deep Skilling Program** conducted by **Cognizant**.

The objective of this program is to strengthen core software engineering concepts, improve problem-solving abilities, and gain practical experience with Java Full Stack development through structured assignments and hands-on implementation.

---

## 🎯 Learning Objectives

- Build a strong foundation in Core Java and Object-Oriented Programming
- Understand SOLID Principles and Design Patterns
- Master Data Structures and Algorithms
- Learn SQL and PL/SQL programming
- Practice Test-Driven Development (TDD)
- Write unit tests using JUnit 5 and Mockito
- Develop clean, maintainable, and scalable code
- Follow industry-standard coding practices using Git and GitHub
